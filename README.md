# Cartooni

이미지를 만화 스타일로 변환합니다. 

---

## 주요 기능

- 트랙바(Trackbar) UI로 실시간 파라미터 조정 가능
- 원본 | 결과 비교 화면 출력 (화면 크기에 맞게 자동 축소)

| 트랙바 | 효과 (값 증가 시) | 권장값 |
|--------|-------------------|--------|
| `Color Flattening` | 색상이 더 단순해짐 | 30 ~ 80 |
| `Edge Detail` | 윤곽선(Edge)이 굵어짐 | 5 ~ 15 |
